# titanusbot

### Setup

`$ cp .env.dist .env`

`$ echo "BOT_TOKEN=your_bot_token_goes_here" > .env`

### Run
`$ ./run.sh`

### See logs

`$ ./logs.sh`

### How to use

After adding the bot to your discord server, type:
`!price bitcoin` or `!price btc`

Change `btc` or `bitcoin` for any cryptocurrency asset you want;

### Stop

`$ ./stop.sh`
